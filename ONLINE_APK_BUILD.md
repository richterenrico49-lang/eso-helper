# ESO Helper - APK online bauen (ohne Computer!)

Diese Anleitung zeigt dir, wie du die APK **direkt von deinem Handy aus** mit einem kostenlosen Online-Service erstellst. **Keine Installation, kein Computer nötig!**

---

## 🚀 Schnelle Übersicht

1. Projekt-Dateien herunterladen
2. Zu einem Online-Build-Service hochladen
3. APK generieren lassen
4. APK auf dein Handy herunterladen
5. Installieren und spielen! 🎮

**Dauer: Ca. 10-15 Minuten**

---

## Schritt 1: Projekt herunterladen

### Option A: Über GitHub (empfohlen)

1. Öffne diese URL in deinem Handy-Browser:
   ```
   https://github.com/moonsugar/eso-helper
   ```

2. Klicke auf den grünen Button **"Code"**

3. Wähle **"Download ZIP"**

4. Die Datei wird heruntergeladen

### Option B: Direkt als ZIP

Wenn du einen direkten Download-Link hast, klicke darauf und speichere die ZIP-Datei.

---

## Schritt 2: Projekt hochladen zu Capacitor Cloud

### Was ist Capacitor Cloud?

Capacitor Cloud ist ein kostenloser Service von Ionic, der deine App automatisch zu einer APK baut.

### Anleitung:

1. **Gehe zu:** https://capacitor.ionicframework.com/

2. **Melde dich an** (kostenlos mit Email oder GitHub)

3. **Neues Projekt erstellen:**
   - Klicke auf **"New App"**
   - Gib einen Namen ein: `ESO Helper`
   - Wähle **"Capacitor"** als Framework

4. **Projekt hochladen:**
   - Klicke auf **"Upload Project"**
   - Wähle die heruntergeladene ZIP-Datei
   - Warte auf den Upload

---

## Schritt 3: APK bauen lassen

1. **Im Capacitor Cloud Dashboard:**
   - Gehe zu deinem Projekt
   - Klicke auf **"Build"**

2. **Wähle Optionen:**
   - Platform: **Android**
   - Build Type: **Release**
   - Klicke **"Start Build"**

3. **Warte auf den Build:**
   - Das dauert ca. 5-10 Minuten
   - Du siehst eine Fortschrittsanzeige
   - Du bekommst eine Benachrichtigung, wenn fertig

---

## Schritt 4: APK herunterladen

1. **Wenn der Build fertig ist:**
   - Klicke auf **"Download APK"**
   - Die Datei wird heruntergeladen

2. **Speichern:**
   - Die APK wird in deinem Download-Ordner gespeichert
   - Merke dir den Namen: `eso-helper-release.apk`

---

## Schritt 5: APK auf dein Handy installieren

### Auf Android-Handy:

1. **Öffne den Dateimanager:**
   - Gehe zu **Downloads**
   - Suche die `eso-helper-release.apk` Datei

2. **Tippe auf die APK-Datei:**
   - Es fragt dich: "Möchtest du diese App installieren?"
   - Klicke **"Installieren"**

3. **Bestätige die Installation:**
   - Wenn es fragt, ob du Apps aus unbekannten Quellen erlaubst
   - Klicke **"Erlauben"** oder **"OK"**

4. **Warte auf die Installation:**
   - Das dauert ein paar Sekunden
   - Wenn fertig, klicke **"Öffnen"**

5. **Fertig!** 🎉
   - Die App startet
   - Du siehst die ESO Helper Startseite

## ❓ Häufig gestellte Fragen

**F: Kostet das etwas?**
A: Nein! Alle Services sind kostenlos für die erste APK.

**F: Wie lange dauert der Build?**
A: Normalerweise 5-15 Minuten, je nach Service.

**F: Kann ich die APK später aktualisieren?**
A: Ja! Einfach die Schritte wiederholen.

**F: Funktioniert die App offline?**
A: Ja, die meisten Features funktionieren offline.

**F: Kann ich die App im Play Store veröffentlichen?**
A: Ja, aber du brauchst ein Google Play Developer Konto (ca. 25€ einmalig).

**F: Was ist, wenn der Build fehlschlägt?**
A: Überprüfe, dass die ZIP-Datei komplett ist. Versuche es mit einem anderen Service.

## 🆘 Probleme?

### APK wird nicht heruntergeladen

- Überprüfe deine Internet-Verbindung
- Versuche einen anderen Browser
- Warte ein paar Minuten und versuche es erneut

### Installation schlägt fehl

- Überprüfe, dass du genug Speicherplatz hast
- Deaktiviere "Unbekannte Quellen" und aktiviere es wieder
- Lösche alte Versionen der App

### App startet nicht

- Starte dein Handy neu
- Deinstalliere die App und installiere erneut
- Überprüfe, dass dein Android-Version aktuell ist
